# Dotfiles for Linux

My personal config/dotfiles files for Linux. In development.

Setup:
- Arch Linux x64
- Urxvt Terminal (AUR package: rxvt-unicode-cvs-patched-wideglyphs)
- i3-gaps
- Polybar
- cli-visualizer
- htop
- Zsh
- Oh-my-zsh with punctual theme
